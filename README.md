# Campus Info Hub

## Project Description
A single-page campus information website designed to demonstrate layout and positioning behaviors using HTML and CSS.

## Layout Behaviors Demonstrated
- Navigation bar with dropdown menu
- Card-based layout for services
- Grid layout for gallery
- Sticky announcements
- Hidden and visible elements
